# banner-compress

Amazon DSP バナー画像をサイズ別の容量上限に収まるよう圧縮するブラウザ完結型ツール。

## 使い方

`index.html` を Chrome / Edge で直接開く（サーバー不要）。

1. フォルダを選択（クリック、またはドラッグ&ドロップ）
2. 「圧縮を開始」を押す
3. 各フォルダ内に `圧縮/` サブフォルダが作成され、圧縮済みファイルが保存される

## 特徴

- ファイル名ではなく画像の実際のピクセルサイズで対象を自動判別
- mozjpeg (WebAssembly) による高画質圧縮（読み込み失敗時はブラウザ標準エンコーダにフォールバック）
- 容量超過時、寸法を保ったまま画質を落として自動リサイズを試みるオプション付き
- 外枠に不自然な白/黒ラインが出ていないかの簡易チェック
- IndexedDBで前回選択したフォルダを記憶

## 配布について

このリポジトリは非公開のため、社外パートナーへは `index.html` を直接ファイル共有（Slack/メール/Drive等）で配布する。
機密情報は含まれていない（クライアントサイド完結・外部送信なし）ため、ファイル単体の共有は問題ない。
